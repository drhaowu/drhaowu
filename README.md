# 吴昊博士的学术主页

基于 Astro 构建的个人学术网站，展示研究成果、项目经历和学术贡献。

## 🚀 技术栈

- [Astro](https://astro.build) - 静态网站生成器
- [Tailwind CSS](https://tailwindcss.com) - 样式框架
- [React](https://reactjs.org) - 交互组件

## 📂 项目结构

```
personal-website/
├── public/              # 静态资源
│   └── img/            # 图片资源
├── src/
│   ├── components/     # 可复用组件
│   ├── layouts/        # 布局模板
│   ├── pages/         # 页面文件
│   └── styles/        # 样式文件
```

## 🛠️ 开发指南

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm run dev
```

3. 构建网站：
```bash
npm run build
```

## 📝 内容更新

- `src/pages/` - 编辑或添加新页面
- `public/img/` - 添加图片等静态资源
- `src/components/` - 添加新的可复用组件
